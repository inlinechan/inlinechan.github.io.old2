This article explains how to install El Capitan to GA-Z77-D3H(rev 1.0/F22 ) with GTX-560ti.

1 Specs

CPU
i7-3770
Motherboard
GA-Z77-D3H (rev 1.0) F22
Graphics
HD4000(built-in i7-3770) / GTX-560ti

2 Detach Graphics

  • Starts with HD4000 first. attach GTX-560ti later

3 Create USB

  1. Download GA-Z77-DS3H pack.
  2. Follow the instructions until step2 from UniBeast: Install OS X El Capitan on Any Supported Intel-based PC
  3. Mount EFI partition by EFI Mounter v3 to put some essential kexts.
  4. Put FakeSMC.kext, NullCPUPowerManagement.kext, and AtherosL1cEthernet.kext into EFI/CLOVER/kexts/10.11
  5. Download El Capitan Post-Install Tools or just Clover_v2.3k_r3320_UEFI.pkg and copy into USB

4 Install

  1. Boot from USB(Press F12 to boot from your USB. UEFI:USB Name)
  2. Do step 4 UniBeast: Install OS X El Capitan on Any Supported Intel-based PC
  3. Press F12 to boot UEFI:USB Name
  4. Select El Capitan

5 Post-install

  1. Install Clover_v2.3k_r3320_UEFI.pkg from your USB to your El Capitan Disk
  2. Mount EFI of El Capitan Disk.
  3. Copy FakeSMC.kext, NullCPUPowerManagement.kext, and AtherosL1cEthernet.kext to EFI/CLOVER/kexts/10.11

6 Enable GTX-560ti

  1. Attach GTX-560ti
  2. Press F12 and enter Options
  3. Graphics -> [+] Inject Nvidia
  4. Select El capitan
  5. Download Clover Configurator.
  6. Mount El Capitan's EFI partition.
  7. Double click config.plist in /Volume/EFI/EFI/CLOVER
  8. Modify config.plist to apply it permanently.

7 Insurance

  • Repeat above steps to get another Mac to recover El Captian in emergency
